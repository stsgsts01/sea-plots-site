как заполнять эту страницу

Папка для этой страницы может выглядеть так:

  • docs/plot-320/presentation/index.html — страница списка презентационных файлов

Такой формат выглядит аккуратно и удобно: посетитель сам выбирает, открыть PDF в браузере или скачать его себе на компьютер.
